# Sistema de Gestión de Reclamos

Aplicación web desarrollada en Python y Flask para registrar y gestionar reclamos.

## Funcionalidades
- Registro de usuarios
- Carga de reclamos
- Seguimiento de estados
- Visualización de información

## Tecnologías utilizadas
- Python
- Flask
- SQLite
- HTML / CSS

## Descripción
Este proyecto fue desarrollado como iniciativa propia para practicar desarrollo backend, manejo de bases de datos y lógica de negocio aplicada a un caso real.
